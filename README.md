## ğŸš€ Getting Started

### Prerequisites

- [Godot 4.5](https://godotengine.org/download) or later
- Basic knowledge of Godot Engine (optional for contributors)

### Opening the Project

1. Clone this repository:
   ```bash
   git clone https://github.com/BrandonDiasDev/exploring-animals.git
   cd exploring-animals
   ```

2. Open Godot 4.5

3. Click "Import" and select the `project.godot` file in this directory

4. Click "Import & Edit"

### Running the Game

- Press `F5` or click the "Play" button in the Godot editor to run the game
- Press `F6` to run the current scene
